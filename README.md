## Asia_BudgetFlights
#### A flight reviews app built with Ruby on Rails and React.js

---
![Home](https://i.imgur.com/mIxmPBW.png)

---
Some of the features of this app include:

* Ruby version: `2.6.3`
* Rails version: `6.0.3.2`
* Database: `PostgreSQL`
* React version: `16.13.1`
* React Hooks API
* React Context API
